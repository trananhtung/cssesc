# Changelog

All notable changes to this project are documented here. The format is based on

## [0.1.0]

### Added

- Initial release: `cssesc` and `cssesc_with` — a zero-dependency, `no_std` port of the
  `cssesc` npm package, with string/identifier modes, quote selection, wrapping, and
  escape-everything.
